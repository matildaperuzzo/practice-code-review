### 🤝 How to Contribute

We are currently suffering from a severe "humor deficit." If you have a joke that’s so good it’s bad (or just actually good), please help us out!

1.  **Fork** this repository.
2.  Add your joke to the bottom of this file.
3.  Open a **Pull Request** with the title `fix: added missing serotonin`.

**Rules:** Keep it clean, keep it clever, and if it makes you groan, it’s probably perfect.
